# 🚀 PhysioAssist Oracle v8.0 - Quick Start Guide

## 📱 الدليل السريع - Quick Start

---

## ✨ **ما الجديد؟**

**بوت واحد ذكي** بدل ثلاثة! 🎯

عند البداية، ستظهر لك **3 أزرار**:

```
🏥 PhysioAssist Oracle v8.0

👨‍💼 Admin Mode
👤 User Mode  
ℹ️ About Modes
```

---

## 🎮 **كيفية الاستخدام:**

### **الخطوة 1️⃣ : ابدأ البوت**

```
1. افتح Telegram
2. ابحث عن: @PhysioAssistBot
3. اضغط /start
```

### **الخطوة 2️⃣ : اختر الوضع**

**اختر أحد الخيارات:**

#### **👨‍💼 Admin Mode** (للمسؤولين)
```
✅ وصول كامل
✅ بدون دفع
✅ تقييمات غير محدودة
✅ رسائل تصحيح الأخطاء
✅ معلومات تفصيلية
```

**متى تختاره:**
- تريد اختبار كل شيء
- تريد رؤية الأخطاء
- تريد تقييمات غير محدودة

#### **👤 User Mode** (للمستخدمين)
```
✅ تقييم مجاني واحد
✅ نظام دفع عامل
✅ خطط اشتراك
✅ تجربة حقيقية
✅ إدارة الاشتراكات
```

**متى تختاره:**
- تريد اختبار نظام الدفع
- تريد التجربة الحقيقية
- تريد اختبار الاشتراكات

#### **ℹ️ About Modes** (معلومات)
```
يعرض لك مقارنة بين الوضعين
```

### **الخطوة 3️⃣ : اختر اللغة**

```
🇬🇧 English
🇸🇦 العربية
```

### **الخطوة 4️⃣ : ابدأ المحادثة**

```
البوت سيسأل عن أعراضك
أجب بطبيعية
البوت سيطرح أسئلة توضيحية
```

### **الخطوة 5️⃣ : التقييم**

```
بعد ~6 رسائل
سيطلب منك تقييم شدة الألم
اختر: خفيف / متوسط / شديد
```

### **الخطوة 6️⃣ : خطة العلاج**

```
ستحصل على خطة شاملة:
✅ نصائح منع التفاقم
✅ علاجات منزلية
✅ برنامج تمارين
✅ تحذيرات
✅ منتجات موصى بها
✅ أدوية آمنة
```

## 🔐 **Admin Mode - التفاصيل**

### **من يستخدمه؟**
- المطورون
- المسؤولون
- فريق الاختبار

### **المميزات:**
- ✅ وصول كامل بدون قيود
- ✅ لا توجد حدود للتقييمات
- ✅ رسائل خطأ مفصلة
- ✅ معلومات تصحيح الأخطاء (Debug)
- ✅ معلومات الجلسة
- ✅ عدد الرسائل
- ✅ استخدام الـ Tokens

### **مثال على الرسالة:**

```
Hello, I have neck pain...

✅ Response from AI...

🐛 **DEBUG INFO:**
Messages: 2
Mode: ADMIN
Tokens: ~45
```

### **متى تستخدمه:**
- ✅ اختبار الميزات الجديدة
- ✅ البحث عن الأخطاء
- ✅ اختبار التحديثات
- ✅ التطوير والصيانة

## 👤 **User Mode - التفاصيل**

### **من يستخدمه؟**
- المستخدمون العاديون
- الاختبار الفعلي
- اختبار نظام الدفع

### **المميزات:**
- ✅ تقييم مجاني واحد
- ✅ نظام دفع حقيقي
- ✅ خطط اشتراك متعددة
- ✅ إدارة الاشتراكات
- ✅ تتبع الصلاحية

### **الخطط المتاحة:**

| الخطة | السعر | المدة | الميزات |
|------|------|------|--------|
| **Basic** | $1.99 | شهر واحد | تقييمات غير محدودة |
| **Premium** | $4.99 | 3 أشهر | + دعم أولوي |
| **Pro** | $9.99 | سنة واحدة | + محتوى حصري |

### **مثال على تدفق الدفع:**

```
1. أكمل التقييم الأول (مجاني)
2. جرّب التقييم الثاني
3. سيطلب منك الترقية
4. اختر خطة
5. ادفع
6. احصل على وصول غير محدود
```

### **متى تستخدمه:**
- ✅ اختبار التجربة الحقيقية
- ✅ اختبار نظام الدفع
- ✅ اختبار الاشتراكات
- ✅ محاكاة المستخدم الفعلي

## 💳 **اختبار الدفع (Sandbox)**

### **بطاقة الاختبار:**

```
رقم البطاقة: 4242 4242 4242 4242
التاريخ: أي تاريخ مستقبلي (مثل 12/25)
CVC: أي 3 أرقام (مثل 123)
```

### **خطوات الدفع:**

```
1. اختر خطة (Basic/Premium/Pro)
2. سيظهر نموذج الدفع
3. أدخل بيانات البطاقة
4. اضغط Pay
5. تم! ✅
```

## 🐛 **معلومات التصحيح (Debug Mode)**

### **في Admin Mode فقط:**

كل رسالة ستحتوي على:

```
🐛 **DEBUG INFO:**
Language: English
Stage: free_chat
Session ID: 8660311978
Messages: 2
Mode: ADMIN
Tokens: ~45
```

### **ماذا تعني؟**

- **Language:** اللغة المستخدمة
- **Stage:** المرحلة الحالية
- **Session ID:** معرّف الجلسة
- **Messages:** عدد الرسائل
- **Mode:** الوضع (Admin/User)
- **Tokens:** عدد الـ Tokens المستخدمة

## 🆘 **حل المشاكل:**

### **المشكلة: البوت لا يستجيب**

**الحل:**
```
1. تحقق من BOT_TOKEN
2. تحقق من الاتصال بالإنترنت
3. أعد تشغيل البوت
4. اكتب /start مرة أخرى
```

### **المشكلة: الدفع لا يعمل**

**الحل:**
```
1. استخدم بطاقة الاختبار
2. تحقق من PAYMENT_PROVIDER_TOKEN
3. جرّب مرة أخرى
```

### **المشكلة: التقييم عالق**

**الحل:**
```
1. اكتب /start
2. ابدأ من جديد
3. تحقق من الأخطاء
```
